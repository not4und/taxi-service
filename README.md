# taxi-service
___
## Project description
___
A simple web-application that supports authentication, registration and other CRUD operations.
## Features
___
* Registration
* Login
* Logout
* Display All Drivers
* Display current driver's Cars
* Display All Cars
* Display All Manufacturers
* Create new Driver
* Create new Car
* Create new Manufacturer
* Add Driver to Car
## Tech stack
___
* JDBC
* Servlet
* JSP
* JSTL
* HTML, CSS
* Apache Tomcat
* MySQL
